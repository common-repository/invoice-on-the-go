Create invoices anywhere (and in seconds) using your phone!

== Description ==

The only wordpress invoice system that has its own android app so you can create invoices in your computer or directly using your phone.

**Features**
* Client, services and invoice management
* Create invoices in your computer or in your phone
* Email the invoice to your client
* PayPal support (so your clients can pay you using your site directly)

**How to use this plugin**
1.- Install this plugin
2.- Install the app "Invoice For WordPress" in your phone: https://play.google.com/store/apps/details?id=com.rednao.invoiceonthego
3.- That's it!




**Features**

*   Create invoices and services.
*   Manage clients
